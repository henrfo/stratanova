Exploring Neural Models of Hippocampal Memory

Computational Neuroscience Memory Systems Neural Networks

Project at Simula Summer School in Computational Physiology (SSCP) investigating how artificial neural networks can model hippocampal memory functions. Implementing and experimenting with Hopfield Networks and Kanerva Machines to simulate pattern completion, pattern separation, and associative memory processes that underlie spatial and episodic memory formation.

Hippocampal Circuit Dynamics in SEEG Data

Electrophysiology Neural Oscillations Memory Circuits

UiO:Life Science summer research project analyzing stereoelectroencephalography (SEEG) data to investigate hippocampal memory circuits and their input regions. Focusing on theta oscillations across the hippocampus and its primary input regions (entorhinal cortex and perirhinal cortex) during novel image processing, this project develops automated pipelines to extract, filter, and analyze neural activity patterns in the medial temporal lobe memory system.
