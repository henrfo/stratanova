Overview

This UiO:Life Science summer research project investigates the dynamic neural circuits underlying human memory formation through analysis of stereoelectroencephalography (SEEG) data. SEEG provides unprecedented access to deep brain structures in humans, allowing us to directly observe hippocampal activity and its interactions with cortical regions during memory processing.

The project focuses on theta oscillations (4-8 Hz), a hallmark of hippocampal function, examining how these rhythms coordinate activity across the medial temporal lobe memory system. By analyzing recordings from patients undergoing clinical monitoring for epilepsy, we gain unique insights into the neural mechanisms of human memory that are impossible to obtain through non-invasive methods.

Methods & Approach

Data Acquisition & Processing

Working with intracranial recordings from epilepsy patients who have electrodes implanted for clinical purposes. These recordings provide millisecond-precision measurements of neural activity from specific brain regions.

  • SEEG recordings: Direct electrical recordings from depth electrodes in hippocampus, entorhinal, and perirhinal cortex
  • Preprocessing pipeline: Artifact removal, line noise filtering, and bad channel detection
  • Anatomical localization: CT/MRI coregistration to precisely identify electrode locations
  • Event detection: Identifying memory-related epochs during image viewing tasks

Signal Analysis Techniques

Developing automated pipelines for extracting and analyzing neural oscillations:

  • Spectral analysis: Fast Fourier Transform and Morlet wavelet decomposition
  • Phase-amplitude coupling: Examining cross-frequency interactions
  • Coherence analysis: Quantifying synchronization between brain regions
  • Information flow: Directed connectivity measures (Granger causality, Transfer Entropy)

Key Research Questions

  • How do theta oscillations coordinate information flow between hippocampus and cortex during memory encoding?
  • What are the temporal dynamics of hippocampal-cortical interactions during novel image processing?
  • How do different hippocampal subfields contribute to memory formation?
  • Can we identify oscillatory biomarkers that predict successful memory encoding?
  • How do pathological oscillations in epilepsy affect normal memory processing?

Technical Implementation

Analysis Pipeline

Pipeline Components:
1. Data Import (EDF/BrainVision formats)
2. Preprocessing (MNE-Python)
3. Time-frequency Analysis (SciPy/Custom)
4. Statistical Testing (Cluster-based permutation)
5. Visualization (Matplotlib/Plotly)

Technologies & Tools

  • Python: Core analysis language with scientific computing stack
  • MNE-Python: Electrophysiology data processing and visualization
  • NumPy/SciPy: Numerical operations and signal processing
  • Pandas: Data management and behavioral analysis
  • Nilearn: Brain imaging and electrode localization
  • HPC clusters: Parallel processing for large datasets

Challenges & Solutions

Working with clinical data presents unique technical and methodological challenges:

  • Data heterogeneity: Standardizing analyses across different electrode configurations
  • Artifact contamination: Developing robust methods to identify and remove epileptic activity
  • Computational demands: Optimizing code for processing hundreds of gigabytes of data
  • Statistical power: Implementing appropriate corrections for multiple comparisons

Neurobiological Insights

Circuit Organization

The project reveals how different components of the medial temporal lobe work together:

  • Hippocampus: Central hub for binding disparate elements into coherent memories
  • Entorhinal Cortex: Primary gateway for cortical inputs to hippocampus
  • Perirhinal Cortex: Object recognition and familiarity processing
  • Theta synchrony: Temporal framework for coordinating multi-region processing

Clinical Relevance

Understanding normal hippocampal function has direct implications for:

  • Epilepsy surgery planning and outcome prediction
  • Early detection of Alzheimer's disease
  • Development of brain-computer interfaces for memory enhancement
  • Understanding memory deficits in psychiatric disorders

Current Progress & Future Directions

Completed Milestones

  • Established preprocessing pipeline for SEEG data
  • Implemented theta detection algorithms
  • Created visualization tools for multi-channel recordings
  • Initial coherence analysis between hippocampus and cortex

Next Steps

  • Machine learning: Developing classifiers to predict memory success from oscillatory features
  • Real-time analysis: Creating tools for online detection of memory-related patterns
  • Cross-species comparison: Relating human findings to animal models
  • Closed-loop stimulation: Designing protocols to enhance memory through targeted stimulation

Collaboration & Impact

This project is part of the UiO:Life Science convergence environment, bringing together expertise from neuroscience, medicine, physics, and computational science. The work contributes to the Human Brain Project and broader efforts to understand human cognition through direct neural recordings.

The automated analysis pipelines developed here will be made openly available, enabling other researchers to analyze SEEG data more efficiently and contributing to reproducible neuroscience research.
